# Brand Assets — bat-id

Déposez ici tous les fichiers SVG, PNG et favicon du projet bat-id.

Claude Code sélectionnera ensuite les bons fichiers pour chaque emplacement du site :
- Logo header (navbar)
- Logo footer
- Favicon (favicon.ico / favicon.svg)
- Logo page de login admin
- Logo emails transactionnels
- Logo factures PDF
- Open Graph / partage réseaux sociaux

## Formats acceptés
- SVG (préféré pour logos)
- PNG (fallback ou Open Graph)
- ICO (favicon legacy)
